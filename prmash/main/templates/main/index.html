Автоматическое управление

  • Описание проекта
    • Проект представляет собой дистанционное управление машинкой. Данное устройство способно отправлять данные на сервер, считывать данные и команды с сервера и выполнять их. С развитием данного проекта можно будет дистанционно управлить не только обычными машинками но и целыми автомобилями. Устройство решает задачу автоматического контроля разного вида техники.
  • принцип работы
    • Arduino UNO передаёт данные с датчиков по Serial к NodeMCU. Конечное устройство обработав данные передаёт их на сервер Django где они тоже обрабатываются и уже только потом выдётся команда(направление куда поехать) обратно к Arduino UNo
